## 📁 Projects

### 📊 [Customer Shopping Behavior Analysis](https://github.com/SuhailDataScience/Data-analysis-project)
End-to-end analysis of **3,900 customer transactions** using Python, SQL, and Power BI.
- Uncovered a ~91% summer revenue drop and flagged that only 2% of customers are new — a critical acquisition gap
- Delivered a Power BI dashboard covering segmentation, category sales, seasonal trends, and shipping analysis
- **Stack:** Python (Pandas, NumPy) · MySQL · Power BI

---

### 🛒 [E-Commerce Store EDA](https://github.com/SuhailDataScience/Ecommerce_Store_EDA)
Comprehensive SQL EDA on the **Olist Brazilian e-commerce dataset** — 99,000+ orders across 8 relational tables.
- Built 10 business-focused analyses: revenue trends, category performance, CLV, payment methods, and RFM segmentation
- Applied CTEs, window functions (`SUM() OVER`), subqueries, and CASE statements throughout
- **Stack:** MySQL · SQL (DDL + DML + CTEs + Window Functions)

---

### 🧹 [Online Retail Data Cleaning & EDA](https://github.com/SuhailDataScience/Datacleaning-Visualisation)
Cleaned a **541,000-row UK retail dataset** down to ~397,884 analysis-ready rows using Pandas.
- Handled nulls, duplicates, cancelled orders, and invalid price/quantity entries
- Built visualizations revealing a November holiday sales spike and the UK's dominant revenue share
- **Stack:** Python · Pandas · Matplotlib · Seaborn · Jupyter Notebook

---

### 📍 [India Sales Performance Dashboard](https://github.com/SuhailDataScience/SalesDashboard-Tableau)
Interactive **Tableau dashboard** analyzing ~18,000 orders of Indian e-commerce data.
- KPI strip, state-wise choropleth map, monthly trend line, and sub-category profit/loss breakdown
- Found that Tables and Chairs sub-categories were consistently loss-making — invisible at the category level
- **Stack:** Tableau · Data Storytelling · Dashboard Design
